Escala de notas

        
Exigencia chilena estándar: 60% del puntaje = nota 4.0. La escala va de 1.0 a 7.0 de forma lineal por tramos. El puntaje máximo depende del número de criterios de la rúbrica activa (3 pts × criterio).

        

          100% → 7.0  |  75% → 5.5  |  60% → 4.0  |  40% → 3.0  |  0% → 1.0

🔗 Sincronización con Google Sheets

        
Cada vez que guardas una evaluación, ArtEval la envía automáticamente a una hoja de cálculo en tu Google Drive. Puedes verla desde cualquier dispositivo y compartirla con otros.

        
1
Ve a sheets.google.com y crea una hoja nueva. Copia la URL completa.

        
2
Ve a Extensiones → Apps Script, borra el contenido y pega el script de abajo. Guarda y despliega como Aplicación web → Cualquier usuario. Copia la URL del despliegue.

        
3
Pega esa URL aquí y presiona Probar.


        
        

          
SCRIPT PARA APPS SCRIPT (copia y pega en tu proyecto):

          
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sh = ss.getSheetByName('ArtEval') || ss.insertSheet('ArtEval');
    if (sh.getLastRow() === 0) {
      sh.appendRow(['Nombre','Curso','Fecha','Rubrica','MB','L','M','N','Pts','MaxPts','Pct','Nota','Resultado']);
      sh.getRange(1,1,1,13).setFontWeight('bold');
    }
    var r = data;
    var res = (r.nota||1) >= 4 ? 'Logrado' : 'No logrado';
    sh.appendRow([r.nombre,r.curso||'',r.fecha,r.rubricaNombre||'',
      r.conteo&&r.conteo.MB||0, r.conteo&&r.conteo.L||0,
      r.conteo&&r.conteo.M||0, r.conteo&&r.conteo.N||0,
      r.pts||0, r.maxPts||0, r.pct||0, r.nota||1, res]);
    return ContentService.createTextOutput(JSON.stringify({ok:true}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({ok:false,error:err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
function doGet(e) {
  return ContentService.createTextOutput(JSON.stringify({ok:true,msg:'ArtEval conectado'}))
    .setMimeType(ContentService.MimeType.JSON);
}

          
        


        
        
        

          
          
          
          
        

        

      


      

        
☁️ Google Drive Sync — Configuración


        
        

        


        
        

          ¿Para qué sirve esto?

          Google Drive Sync guarda automáticamente todas tus evaluaciones en la nube. Si cambias de computador, reinstalas el navegador o usas ArtEval desde otro dispositivo, tus datos estarán seguros y podrás recuperarlos con un clic.


          ¿Es complicado?

          Necesitas crear un "permiso de acceso" gratuito en Google Cloud Console. El proceso toma unos 10 minutos y solo se hace una vez. Sigue los pasos de abajo.
        


        
        


          
          

            
1

            

              
Abre Google Cloud Console

              
Haz clic en el botón de abajo. Se abrirá una nueva pestaña. Si no tienes cuenta de Google Cloud, te pedirá crearla — es gratis.

              
                Abrir Google Cloud Console →
              
            

          


          
          

            
2

            

              
Crea un proyecto nuevo

              
En la barra superior, haz clic en el selector de proyectos (donde dice "Seleccionar proyecto") → Proyecto nuevo → ponle el nombre "ArtEval" → Crear.

              
💡 Si ya tienes un proyecto tuyo, puedes usarlo directamente.

            

          


          
          

            
3

            

              
Activa la API de Google Drive

              
En el menú izquierdo ve a APIs y servicios → Biblioteca. Busca "Google Drive API", haz clic en ella y presiona Habilitar.

              
                Ir a habilitar Google Drive API →
              
            

          


          
          

            
4

            

              
Configura la pantalla de consentimiento OAuth

              
Ve a APIs y servicios → Pantalla de consentimiento de OAuth → selecciona Externo → Crear. Completa solo los campos obligatorios: nombre de la app (ArtEval), correo de soporte y correo del desarrollador. Guarda y continúa hasta llegar a Publicar app.

              
                Ir a pantalla de consentimiento →
              
            

          


          
          

            
5

            

              
Crea las credenciales OAuth 2.0

              
Ve a APIs y servicios → Credenciales → + Crear credenciales → ID de cliente de OAuth. En tipo de aplicación selecciona Aplicación web. Ponle nombre "ArtEval Web".

              
En "Orígenes de JavaScript autorizados" agrega la dirección donde usas ArtEval:

              

                
Tu origen actual (agrégalo en Google Cloud):

                

                  
                  
                

                

              

              
                Ir a crear credenciales →
              
            

          


          
          

            
6

            

              
Copia el Client ID y pégalo aquí

              
Después de crear las credenciales, aparecerá una ventana con el ID de cliente. Cópialo y pégalo en el campo de abajo. Tiene el formato xxxx.apps.googleusercontent.com

              
              
              

                
                
              

            

          


        


        
        

          ¿Problemas o preguntas frecuentes?
          

            
❓ "Error: origen no autorizado"
Ve a las credenciales en Cloud Console y agrega exactamente la URL que aparece en el Paso 5 en "Orígenes autorizados".

            
❓ "Error: acceso denegado"
Asegúrate de haber completado la pantalla de consentimiento OAuth (Paso 4) y de haber habilitado la API de Drive (Paso 3).

            
❓ Uso ArtEval como archivo HTML local (sin servidor)
Los archivos locales tienen restricciones de seguridad en Google OAuth. Te recomendamos publicar ArtEval en Netlify (gratis) para usar Drive Sync. Ver la sección "Publicar en Netlify" en este mismo panel.

            
❓ ¿Mis datos son privados?
Sí. ArtEval guarda los archivos en una carpeta llamada ArtEval_Backups en tu propio Google Drive personal. Solo tú tienes acceso.

          

        


      


      

        
Datos y privacidad

        
Todos los registros se guardan en el almacenamiento local de tu navegador (localStorage). La sincronización con Google Sheets es opcional y adicional al almacenamiento local.
